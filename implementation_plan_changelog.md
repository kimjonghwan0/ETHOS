# 구현 계획 변경 로그 (implementation_plan_changelog.md)

## [2026-03-13 22:45] 세션 통합 변경 내역 (v3 → v3.1)

### [1차] 시나리오 및 시스템 프롬프트 전면 개편
- **시나리오 교체**: 기존 5개 시나리오를 새 5개(이현도·박서연·김민석·정태영·오지현)로 완전 교체 및 하드코딩
- **시스템 프롬프트 리뉴얼**: 감정 상태, 반응 분기, 키워드 바스켓 방식 도입 및 T3.5 단계 추가
- **UI 개선**: Turn Tracker에 T3.5 pill 추가 및 결과 화면(윤리 포인트, KISDI 연결) 강화

### [2차] 안정성 및 오류 수정
- **로그인 체크**: `checkAuth` 미정의 시 안전 처리 추가
- **이미지 오류**: 프로필 이미지 로드 실패 시 `display:none` 처리 및 초기값 설정

### [3차] 페르소나 카드 정보 강화
- **소개란 추가**: 페르소나 카드에 성격 유형 외에 '소개' 섹션 추가하여 몰입도 향상

### [4차] 상황 소개 가독성 및 비디오 로딩 문제 해결
- **상황 카드 개선**: 상황 설명을 인용구(`quote`)와 상세 설명(`desc`)으로 분리하여 가독성 증대
- **비디오 로딩 문제 해결**: 파일명의 한글/특수문자 인코딩 문제 해결을 위해 `video1.mp4`~`video6.mp4`로 파일명 단순화 및 `video.html` 경로 업데이트

## [2026-04-11 11:46] API 연동 및 실행 오류 해결 (v3.1 → v3.2)

### [1차] API 초기 설정 및 크래시 방지
- **환경 변수 로드 최적화**: `load_dotenv()` 위치를 최상단으로 이동하여 API 키 로드 보장
- **테스트 코드 제거**: API 키가 로드되기 전에 잘못된 모델명(`gemini-3`)으로 호출을 시도하는 상단 테스트 블록 제거
- **라이브러리 정리**: `google-genai`와 `google-generativeai` 라이브러리 혼용으로 인한 import 충돌 해결 (`google-generativeai`로 단일화)

### [2차] 모델 설정 및 안정화
- **모델명 수정**: 존재하지 않는 `gemini-3-flash-preview`를 `gemini-1.5-flash`로 수정하여 호출 성공 보장
- **중복 코드 정리**: `API_KEY` 환경 변수 취득 시의 중복 로직 정리

## 파일 구조 변경 및 정리
